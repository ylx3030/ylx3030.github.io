hexo和markdown使用

*hexo

简单发布文章
Hexo根目录下打开cmd终端命令
hexo new post xxxx
本地查看预览效果
hexo s
部署到git上
hexo clean && hexo g && hexo d
hexo clean 清理本地缓存
hexo g generate，生成静态html文件。
hexo d deploy 部署到Coding上。

*markdown

#表示表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。
“```”包裹一段代码，并指定一种语言,可以描写代码
制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行
